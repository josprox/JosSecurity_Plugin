== Description ==

El plugin JosSecurity WP agrega todas las funciones de JosSecurity a tu sitio WordPress, incluyendo la protección contra ataques, la adición de elementos en el encabezado y pie de página, y mucho más.

== Installation ==

1. Descarga el plugin de https://github.com/josprox/JosSecurity_Plugin.
2. Sube el archivo .zip descargado a tu sitio WordPress en la sección "Plugins > Añadir nuevo".
3. Haz clic en "Instalar ahora".
4. Una vez instalado, haz clic en "Activar".

== Frequently Asked Questions ==

= ¿Cómo puedo configurar el plugin? =

Para configurar el plugin, ve a la sección "Ajustes > JosSecurity WP" en el panel de administración de WordPress. Allí encontrarás opciones para configurar diferentes aspectos del plugin.

 Este plugin no cuenta con ninguna versión premium.
